Project 1

The premise of my first group project was to develope a website that pulls information from more than two API's and displays the information on the website. We used National Parks services (NPS), open weather map API, and pexels image API. The idea being, a user can pick a state and a list of National parks and monuments will appear where the user can click and find out more information.

Day Planner

This is a simple day planner app. The user can enter things to accomplish within 9am and 5pm. The user input is stored in local memory so information isnt lost when the page is refreshed. Also, for every hour that passes the color will change from green to red indicating you better have finished what you needed to do.
